Чтобы добавить ненумерованные списки, необходимо пункты выделить звёздочкой (*).
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Напримет, вот так:
1. Первый пункт
2. Второй пункт
